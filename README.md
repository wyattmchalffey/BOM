# Battles of Masadoria

Digital prototype of the card game Battles of Masadoria (LÖVE 2D / Lua).

## Run the game

See **[love2d/README.md](love2d/README.md)** for setup and how to run (e.g. `.\love2d\run.ps1` or drag the `love2d` folder onto LÖVE).

## Project layout

- **love2d/** — LÖVE game (entry point, game state, UI, card definitions).
